* 使用pandas完成文件乱序
```python
	import pandas as pd
	# 如果是中文文件名或内容
	df = pd.read_csv("vocab_converted.csv", encoding="utf-8")  
	# 抽取百分之百的样本数据打乱并且重置索引
	df_shuffled = df.sample(frac=1, 
		random_state=42).reset_index(drop=True)

	# 重新写入文件设置编码并且不将df的索引写入到文件中
	df_shuffled.to_csv("vocab_converted_shuffled.csv", index=False, 
		encoding="utf-8")

```

* 给单词加上英标
```python
import pandas as pd  
import eng_to_ipa as ipa  
  
# 读取你的CSV  
df = pd.read_csv("vocab_converted_shuffled.csv")  
  
# 假设第一列是单词，自动识别列名  
word_col = df.columns[0]  
  
# 为每个单词生成音标，新增一列  
df["phonetic"] = df[word_col].apply(  
    lambda w: ipa.convert(str(w).strip()) if pd.notna(w) else ""  
)  
  
# 保存为新文件  
df.to_csv("vocabulary_with_pronunciation.csv", index=False, encoding="utf-8-sig")  
  
print("完成！前5行预览：")  
print(df.head())
```
