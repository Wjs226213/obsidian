## 3 网络基础
### 3.1 常见网络设备
- 集线器
- 交换机：作用——用于将多个服务器进行连接，组建局域网络
- 路由器：作用——用于实现多个网络之间的互通

### 3.2 网络传输的介质
- 双绞线
  - 就是日常所见的网线
  - 标准
    - T568A：绿白1，绿2，橙白3，蓝4，蓝白5，橙6，棕白7，棕8
    - T568B：橙白1，橙2，绿白3，蓝4，蓝白5，绿6，棕白7，棕8
  - 两种类型
    - 直通线：双绞线的两端标准一致，主要用于连接不同设备
    - 交叉线：双绞线的两端标准不一致，主要用于连接相同设备
- 光纤

### 3.3 网络通信的三要素
- IP：用于指向网络范围内的某一个网络设备
- 端口：用于标识在服务器中唯一的应用程序（服务）
- 协议：用于表示网络传输过程中的规则或者格式（双方统一通信规则）
- 注意：当需要访问服务器上某一个服务（应用程序），务必要知道：设备IP、服务的端口号、通信协议
